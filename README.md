# Sock Security System
### *Знай свои носки в лицо!*

Этот проект поможет вам в удобной форме мониторить свои носки, выбрать лучшую пару на предстоящий день и не пропустить стирку! Данный сайт поможет всем носителям носков облегчить свою ежедневную ношу. На этом сайте вы можете добавлять свои носки, отслеживать их чистоту, смотреть статистику и многое другое.

Ввиду поздней демонстрации идеи (12.12), зачетной недели и скорого экзамена (26.12), находится в разработке, но кое-что уже есть

Для локального запуска скачайте данный репозиторий локально, установите Python3 на свой ПК, выполните `npm run build && npm start` в папке `react/` и `python3 app.py` в `flask/` (Запуск должен быть паралельным). Сайт откроется на `127.0.0.1:3000`
В данный момент на сайт можно попасть по [ссылке](https://socks-security-system-react.up.railway.app/)*

Для добавления носков нажмите кнопку "Добавить носок" в верхней части страницы, заполните информацию о нем и нажмите кнопку "+ Добавить носок" внизу страницы. Чистота носков отображается в таблице на главной странице. Нажмите "Испачкать"/"Постирать" чтобы обновить статус носка.

В проекте фронт и бек выделены в отдельные папки (`react/` и `flask/` соотвественно). Бек состоит из `app.py` -- апи сайта, и `requirements.txt`. Фронт построен по стандарту React приложений ~~(да спасет бог его душу)~~: По одному файлу на `html` (хедр+точка входа), `css`, `tsx` (На момент написания последнего обновления этого описания в одном файле) и все такое.
