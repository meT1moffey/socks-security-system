# Sock Security System
### *Знай свои носки в лицо!*

Этот проект поможет вам в удобной форме мониторить свои носки, выбрать лучшую пару на предстоящий день и не пропустить стирку! Данный сайт поможет всем носителям носков облегчить свою ежедневную ношу. На этом сайте вы можете добавлять свои носки, отслеживать их чистоту, смотреть статистику и многое другое.

Ввиду поздней демонстрации идеи (12.12), зачетной недели и скорого экзамена (26.12), находится в разработке, но кое-что уже есть

Для локального запуска скачайте данный репозиторий локально, установите Python3 на свой ПК, выполните `npm install` и запустите app.py. Сайт откроется на `127.0.0.1:5000`
В данный момент на сайт можно попасть по [ссылке](https://socks-security-system-react.up.railway.app/)*

Для добавления носков нажмите кнопку "Добавить носок" в верхней части страницы, заполните информацию о нем и нажмите кнопку "+ Добавить носок" внизу страницы. Чистота носков отображается в таблице на главной странице. Нажмите "Испачкать"/"Постирать" чтобы обновить статус носка. Обратите внимание, что чистые носки отображаются выше грязных.

В проекте испоьзуется стандартная структура flask приложений: страницы в templates/, стили в static/ ит.д.
